# Problems

These are the shared problems for Bunnings APIs. All problems are compliant with [RFC 7807](https://tools.ietf.org/html/rfc7807); some include extensions to the base `application/problem+json` media type. Refer to individual pages for schema definitions.

Problems returned in this format include a Problem Type URL that can be used to interpret the rest of the response. The URL is of the format `https://problem.api.bunnings.com.au?type=<problemName>`, for example: `https://problem.api.bunnings.com.au?type=invalidServerState`.
